# Kinematics – 3D Motion Analysis

## Problem
r(t) = (a cos(ωt), b sin(ωt), bt)

---

## (a) Trajectory Equation

x = a cos(ωt)  
y = b sin(ωt)

=> (x² / a²) + (y² / b²) = 1  

z = bt  

Trajectory: Elliptical Helix

---

## (b) Path Length

v(t) = dr/dt  
= (-aω sin(ωt), bω cos(ωt), b)

|v(t)| = √(a²ω² sin²(ωt) + b²ω² cos²(ωt) + b²)

L = ∫₀^{t₀} √(a²ω² sin²(ωt) + b²ω² cos²(ωt) + b²) dt

---

## Special Case (a = b = R)

|v| = √(R²ω² + R²)

L = t₀ √(R²ω² + R²)

---

## Interpretation

- Projection on xy-plane: ellipse  
- z increases linearly with time  
- Full motion: helical trajectory  

---

## Special Cases

- a = b → circular helix  
- b = 0 → straight line along x  
- ω = 0 → linear motion in z  
- large ω → tighter spiral
