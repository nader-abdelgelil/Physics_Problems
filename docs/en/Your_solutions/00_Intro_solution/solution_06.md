# Solution 06: Function Analysis (Calculus & Derivatives)

## 1. Problem Statement
Given the function:
$$f(x) = 3x^2 + 2x - 5$$
Find the **derivative** $f'(x)$ and calculate the **slope** of the tangent line at $x = 2$.

---

## 2. Mathematical Steps

### Step 1: Find the Derivative ($f'(x)$)
We use the **Power Rule** for differentiation: 
If $f(x) = ax^n$, then $f'(x) = n \cdot ax^{n-1}$.

Applying this to our function:
1.  For $3x^2$: $2 \cdot 3x^{2-1} = 6x$
2.  For $2x$: $1 \cdot 2x^{1-1} = 2$
3.  For $-5$: The derivative of a constant is always **0**.

**Resulting Derivative:**
$$f'(x) = 6x + 2$$

### Step 2: Calculate the Slope at $x = 2$
The derivative represents the slope at any point. To find the specific slope at $x = 2$, substitute 2 into $f'(x)$:
$$f'(2) = 6(2) + 2$$
$$f'(2) = 12 + 2$$
$$\mathbf{Slope = 14}$$

---

## 3. Physical Meaning (Why we do this?)
In physics, if $f(x)$ represents the **Position** ($s$) of an object over **Time** ($t$), then the derivative $f'(x)$ gives you the **Instantaneous Velocity** ($v$) at that exact second.

At $x = 2$, the "velocity" of this function is $14 \text{ units/sec}$.

---

## 4. Key Concepts
* **Derivative ($f'$)**: The rate of change or the slope of the tangent line.
* **Power Rule**: The shortcut used to solve polynomials easily.
* **Tangent Line**: A straight line that touches the curve at exactly one point.
* - To find the slope at $x=2$, we plug the value into the derivative:
- $f'(2) = 6(2) + 2$
- $f'(2) = 12 + 2 = 14$
- This value (14) represents the instantaneous rate of change.
