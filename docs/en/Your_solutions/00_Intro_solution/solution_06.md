# Function Analysis – Local Extrema

## Problem
Analyze the function:

f(x) = 3x^2 - 12x + 7

Identify any local maxima or minima.

---

## Step 1: Compute the first derivative

f'(x) = 6x - 12

---

## Step 2: Find critical points

Set derivative to zero:

6x - 12 = 0

x = 2

---

## Step 3: Determine the type of critical point

Compute the second derivative:

f''(x) = 6

Since f''(x) > 0 → the function is concave up

➡️ Therefore, x = 2 is a local minimum

---

## Step 4: Find the minimum value

f(2) = 3(2^2) - 12(2) + 7  
f(2) = 12 - 24 + 7 = -5

---

## Final Answer

- Local Minimum at: x = 2  
- Minimum Value: f(2) = -5  

- No local maximum (parabola opens upward)
