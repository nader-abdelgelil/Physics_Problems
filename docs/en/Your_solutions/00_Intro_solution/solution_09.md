# Solution 09: Average Speed vs. Average Velocity

## 1. Problem Statement
An object moves from point **A** to point **B** (100 meters East) in 10 seconds, then turns around and moves to point **C** (40 meters West) in 5 seconds.
Calculate:
1. **Average Speed**
2. **Average Velocity**

---

## 2. Key Definitions
- **Distance ($d$)**: Total path traveled (Scalar).
- **Displacement ($\Delta x$)**: Straight-line distance from start to end (Vector).
- **Average Speed**: $\frac{\text{Total Distance}}{\text{Total Time}}$
- **Average Velocity**: $\frac{\text{Displacement}}{\text{Total Time}}$

---

## 3. Step-by-Step Calculation

### Step 1: Calculate Total Distance and Displacement
- **Total Distance ($d$)**: $100\text{ m} + 40\text{ m} = \mathbf{140\text{ m}}$
- **Displacement ($\Delta x$)**: $100\text{ m} - 40\text{ m} = \mathbf{60\text{ m (East)}}$
- **Total Time ($t$ )**: $10\text{ s} + 5\text{ s} = \mathbf{15\text{ s}}$

### Step 2: Calculate Average Speed
$$\text{Average Speed} = \frac{140\text{ m}}{15\text{ s}} \approx \mathbf{9.33\text{ m/s}}$$

### Step 3: Calculate Average Velocity
$$\text{Average Velocity} = \frac{60\text{ m}}{15\text{ s}} = \mathbf{4.0\text{ m/s (East)}}$$

---
