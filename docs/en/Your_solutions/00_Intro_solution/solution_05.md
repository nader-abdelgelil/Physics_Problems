# Solution 05: Trigonometry (Vector Components)

## 1. Problem Statement
A vector $\vec{A}$ has:
- **Magnitude**: $|\vec{A}| = 15$
- **Angle**: $\theta = 60^\circ$ (with the horizontal axis)

We need to calculate its horizontal component ($A_x$) and vertical component ($A_y$).

---

## 2. Mathematical Formulas
When a vector is at an angle, we use sine and cosine to find its components:
1. **Horizontal Component ($A_x$)**: $A_x = |\vec{A}| \cos(\theta)$
2. **Vertical Component ($A_y$)**: $A_y = |\vec{A}| \sin(\theta)$

---

## 3. Step-by-Step Calculation

### Step 1: Calculate the Horizontal Component ($A_x$)
Using the cosine function:
$$A_x = 15 \cdot \cos(60^\circ)$$
Since $\cos(60^\circ) = 0.5$:
$$A_x = 15 \cdot 0.5 = \mathbf{7.5}$$

### Step 2: Calculate the Vertical Component ($A_y$)
Using the sine function:
$$A_y = 15 \cdot \sin(60^\circ)$$
Since $\sin(60^\circ) = \frac{\sqrt{3}}{2} \approx 0.866$:
$$A_y = 15 \cdot 0.866 = \mathbf{12.99}$$

---

## 4. Final Result
The vector $\vec{A}$ can be written in component form as:
$$\vec{A} = [7.5, 12.99]$$

---

## 5. Verification
To verify, we can use the Pythagorean theorem:
$$|\vec{A}| = \sqrt{A_x^2 + A_y^2}$$
$$|\vec{A}| = \sqrt{7.5^2 + 12.99^2} \approx \sqrt{56.25 + 168.74} = \sqrt{224.99} \approx 15$$
The result is correct.
